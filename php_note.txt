# PHP Syntax, Variables & Data Types

## Html within Php
    <?php $hour = date("H") ?>

    <?php
    if ($hour < 6 || $hour > 18) {
        echo "<b>Night Time.</b>";
    } else {
        echo "<i>Day Time.</i>";
    }
    ?>    

## Php within Html
    <p>
        <?php $hour = date("H") ?>
        <?php if ($hour < 6 || $hour > 18) { ?>
            <b>Night Time.</b>
        <?php } else { ?>
            <i>Day Time.</i>
        <?php } ?>
    </p>
